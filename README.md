# Improving Collaboration by Integrating Multiple Tools

##  Project Overview
This project improves team collaboration by integrating multiple tools into a single automated workflow.
It connects Slack, Flask, GitHub and Trello to reduce manual work and improve productivity.

##  Features
- Slack Bot Notifications
- Flask Backend Integration
- GitHub Repository Automation
- Automated Workflow

## Technologies Used
- Python
- Flask
- Slack API
- GitHub API
- Postman (API Testing)

##  Project Workflow
1. User sends command in Slack
2. Flask backend processes the request
3. APIs communicate with GitHub 
4. Automated notifications are sent to team members
